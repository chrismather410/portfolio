Latest Project

Forecaster

This has been my first major personal project. The app retrieves weekly forecasts for a given location and makes use of the OpenWeather and Geoapify API's. The app has been built in React using Vite as the build process and Chakra UI as opposed to styled components. The main features of this app included light / dark mode, address autocomplete, weekly forecasts and forecast highlights. Forecaster can be found on my GitHub along with instructions after cloning.

Technologies used include:

  • React
  • TypeScript
  • Context API
